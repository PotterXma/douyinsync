# Planning Artifacts（BMAD）

本目录为 **DouyinSync** 的 BMAD 规划层产出，与 `docs/` 技术文档互补：

| 文件 | 说明 |
|------|------|
| [product-brief.md](./product-brief.md) | 产品简报（问题 / 价值 / 范围） |
| [PRD-snapshot.md](./PRD-snapshot.md) | 产品需求快照（从 README/架构提炼） |
| [epics.md](./epics.md) | 指向 `epics-and-stories.md` 的命名入口 |
| [epics-and-stories.md](./epics-and-stories.md) | Epic / Story 全量分解与验收标准 |
| [epics/index.md](./epics/index.md) | 分片 Epic 导航索引 |
| [epic-1.md](./epic-1.md) … [epic-5.md](./epic-5.md) | 分片 Epic 摘要（兼容 `*epic*.md` 发现） |
| [traceability-matrix.md](./traceability-matrix.md) | 功能需求 → Epic → 代码/测试 追溯 |
| [implementation-readiness.md](./implementation-readiness.md) | 实现就绪核对（IR 摘要） |
| [RETROSPECTIVES.md](./RETROSPECTIVES.md) | 全 Epic 回顾（与 sprint 状态一致） |
| [prd-per-video-upload-progress-ux.md](./prd-per-video-upload-progress-ux.md) | **PRD**：单视频上传/管道进度可感知（待拆 Story） |

配置入口：`_bmad/bmm/config.yaml`。

**与工程文档的衔接**：[docs/bmad-documentation-alignment.md](../../docs/bmad-documentation-alignment.md)（BMAD `document-project` 技能链、`docs/` 与 `_bmad-output/` 分工）；[docs/index.md](../../docs/index.md)（总索引）。
